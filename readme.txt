Objective :-

IPL Dashboard
An interactive dashboard built using Streamlit and Plotly to analyze the IPL 2024 dataset. The dashboard provides insights into team performances, including total runs, wickets taken, and other key metrics.

Tools and Libraries Used :-

Python: For data analysis and dashboard development.
Streamlit: To create the interactive web-based dashboard.
Plotly: For visualizing data with interactive charts.
Pandas: For data manipulation and preprocessing.

Data Source :- 

1. www.kaggle.com
2. www.iplt20.com

Execution Steps :-

1. Install Required Software

    Ensure Python 3.x is installed on your system. You can download it from python.org.
    Install a code editor like VS Code or use an IDE like PyCharm.

2. Clone the Repository

    Open a terminal or command prompt.
    Run the following command to clone the repository to your local machine:
	
	git clone https://github.com/KushSaraf/Team_Arceus_Sports_Analytics_Dashboard

    Navigate to the project folder:

    	cd src

3. Install Dependencies

    Install all the necessary Python packages.

5. Run the Dashboard Script

    Launch the IPL Dashboard by running the main Python file:

    streamlit run finalcode.py

    This will start a local server, and the terminal will display the URL (e.g., http://127.0.0.1:8050/).

6. Open the Dashboard

    Open a web browser and go to the displayed URL (usually http://127.0.0.1:8050/).
    Explore the interactive visualizations and features of the IPL Dashboard.

7. Explore the Features

    Use dropdowns, filters, and charts to analyze IPL data interactively.
    Test the performance metrics, player statistics, and match trends as per your interest.
